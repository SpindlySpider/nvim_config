 - gcc - for mason 
 - [sharkdp/fd (finder)](https://github.com/sharkdp/fd)
 - [nerd fonts](https://www.nerdfonts.com/)
 - [BurntSushi/ripgrep](https://github.com/BurntSushi/ripgrep)
